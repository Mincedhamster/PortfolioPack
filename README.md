## Безшкурная Д.С.

#### Для работы сборки у вас должны быть установлены
* node.js
* npm 5
* gulp-cli
* webpack


#### Как установить проект
1. clone this rep
2. npm i
3. gulp

#### Структура проекта
build - готовая папка проекта
* assets
	* styles
	* scripts

src - каталог разработчика
* images
* scripts
	* common
* styles		
	* common
* templates	
	* common
	* pages
